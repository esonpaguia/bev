# Sample of ng-route

## Prerequisite
* *Node.js* - <a href="http://nodejs.org/download/">Download</a> and Install Node.js

## Setup
* Install node packages
  ```bash
  $ cd bev
  $ npm install
  ```

## Run
* Run application
  ```bash
  $ node server
  ```
* Then, open a browser and go to:
  ```bash
  http://localhost:3000
  ```
